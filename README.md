# NeuroLang Interpreter

Welcome to **NeuroLang**, a cutting-edge **AI-powered programming language** inspired by my Neurova project! 🚀  

NeuroLang is designed to make **learning programming interactive, fun, and educational**, while giving you a taste of AI-powered logic. This repository contains a **step-by-step interpreter built with Kivy**, where you can run your NeuroLang code and watch it execute line by line — just like having a coding tutor at your fingertips.

---

## 🔹 Features

- **Core Commands**: Use `say`, `remember`, `if bigger_or_equal then`, `repeat N times`, and `create function ... end`.
- **Step-by-Step Execution**: Watch each line of your code execute, simulating AI thinking.
- **Variables & Memory**: Track variables across your scripts and use them in conditions and loops.
- **Gamified Learning**: Supports quizzes, notes, and premium logic simulation for interactive learning.
- **Easy to Extend**: Add new functions, commands, or AI interactions as you grow your skills.
